** Virtusa Mini Projects**

This repository contains three mini projects completed as part of Virtusa training.

## Projects

### Java – FinSafe Transaction Validator
A digital wallet system that validates transactions and prevents overdraft errors.

### Python – OpsBot Log Automator
A log processing script that filters errors and generates security alerts.

### SQL – Retail Insights
A data analysis project that identifies expiring products, dead stock, and revenue contribution.

## Technologies Used
- Java
- Python
- MySQL
